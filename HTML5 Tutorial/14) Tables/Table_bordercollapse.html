Table Programming in HTML 

An HTML table is defined with the table tag.

Each table row is defined with the tr tag. A table header is defined with the  tag. By default, table headings are bold and centered. 
A table data/cell is defined with the td tag.
Note: The  elements are the data containers of the table.
They can contain all sorts of HTML elements; text, images, lists, other tables, etc.
