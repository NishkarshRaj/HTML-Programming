Link Color is of Four Types:
1) Link => Color of link => default is blue 
2) Hover => Color of link when mouse is upon it => default is blue with underline
3) Active => Color when link is on => default is red and underlined 
4) Visited => Color when link has been visited in the past => default is purple and underlined

Three Things define type of link: 1) Color 2) Background 3) Underline (or lack of)

They can be improvized using CSS Style tag as 1) Internal and 2) External but not as Inline
